##InformationReader 掌中小报##

多频道选择，资讯类阅读，类似今日头条、腾讯新闻等主流新闻客户端软件

### 声明 ###

**掌中小报**内容经过简化，由个人开发者提供，并不用于任何商业用途，仅供个人学习和交流使用。
